# Mockup Templates

This folder contains t-shirt template files for Phase 2 mockup generation.

## Required Templates:

1. **flat-lay-black.png** - Flat lay t-shirt on black background
2. **model-front-black.png** - Model wearing black t-shirt (front view)
3. **folded-black.png** - Folded black t-shirt
4. **lifestyle-black.png** - Lifestyle shot with black t-shirt

## Template Sources:
- Freepik (free tier)
- Mockey.ai (free templates)
- Pixabay (free images)
- Create your own with AI image generation

## Specifications:
- Resolution: 1200x1200px minimum
- Format: PNG with transparency
- Black t-shirts on neutral/white backgrounds work best
- Leave clear area for design overlay (defined in API)

## Region Coordinates:
The API will overlay designs at these positions:
- Flat lay: x=200, y=300, width=600, height=600
- Model front: x=180, y=280, width=640, height=640
- Folded: x=150, y=200, width=500, height=500
- Lifestyle: x=220, y=320, width=560, height=560
